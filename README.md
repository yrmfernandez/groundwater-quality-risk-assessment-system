Groundwater Quality Risk Assessment System

An end-to-end machine learning and web application for assessing groundwater quality for irrigation and drinking purposes.

Project Goals

- Predict irrigation suitability
- Predict drinking water suitability
- Detect groundwater anomalies
- Explain predictions using Explainable AI (SHAP)
- Deploy an easy-to-use web application

Tech Stack

Machine Learning
- Python
- Scikit-learn
- XGBoost
- Isolation Forest
- SHAP

Backend
- Flask

Frontend
- React
- Tailwind CSS

Database
- SQLite / PostgreSQL
